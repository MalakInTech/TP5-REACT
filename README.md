# TP React Débutant – Formulaires et State

## 📌 Description

Ce TP m'a appris à créer une application React simple pour :

- Créer des formulaires contrôlés et non-contrôlés
- Partager l’état entre composants (lifting state up)
- Utiliser le contexte pour partager des données globales
- Gérer des événements et le rendu conditionnel

---

## 🛠 Technologies utilisées

- React.js
- JavaScript ES6
- HTML / CSS

## 🧪 Fonctionnement

- **Formulaire contrôlé** : les champs sont liés au state (`FormulaireControle.js`)  
- **Formulaire non-contrôlé** : les valeurs sont lues via `useRef` (`FormulaireNonControle.js`)  
- **Lifting state up** : `TemperatureConvertor.js` et `TemperatureInput.js` montrent comment un enfant informe le parent  
- **Contexte utilisateur** : `Profil.js` utilise `UtilisateurContext` pour afficher le nom et l’état de connexion
